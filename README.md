Tailwind is a utility first approach
Tailwind Installation and SetUp
Install Tailwind CSS Intellisense from extensions
Create folder and generate package.json by:
npm init -y
Run: npm install tailwindcss @tailwindcss/cli
Create a style.css file and import: @import "tailwindcss"
Run the command: npx @tailwindcss/cli -i ./style.css -o ./output.css --watch
--it means it takes input from style.css and give output by creating the output.css in watch mode
